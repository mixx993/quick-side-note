# 更新记录

## v1.5.2 - 2026-07-10

- 修复默认和最小窗口尺寸下底部“保存、清空、隐藏、设置”操作栏不可见的问题。
- 所有设置页改为独立滚动内容区，设置窗口可调整大小并限制在当前显示器工作区内。
- 新增 OCR/翻译任务 HUD，显示框选、识别、翻译、完成和失败状态；翻译中可通过侧键或 Esc 取消。
- 失败状态提供重新框选和返回便签入口，取消或失败后恢复任务开始前的窗口可见状态。
- 清空便签页改为 8 秒撤销流程，避免误操作立即覆盖文本文件。
- 新增侧键响应模式：兼容模式保留双击窗口切换，即时框选模式消除单击等待。
- API 页面新增连接测试、状态提示和“清除并保存”确认；浏览器快捷键拦截文案与真实行为一致。
- 页签增加更多菜单和键盘重命名入口，主要操作控件支持键盘聚焦。
- 调整暖色主题的辅助文字、强调文字和成功状态对比度。

## v1.5.1 - 2026-07-10

- 便签和状态文件改为原子保存，并保留最近一次备份用于恢复。
- 输入后执行防抖自动保存，状态栏显示未保存、保存中、已保存或保存失败。
- 侧键和浏览器键钩子失败后使用退避重试，避免持续创建后台线程。
- OCR/翻译任务改为主线程事件队列回收；翻译中可再次按侧键取消，退出后不再访问已销毁窗口。
- Windows OCR 失败时默认不上传截图；图片兜底改为设置页明确授权的可选行为。
- 日志不再记录 OCR 原文，增加大小限制和轮转。
- 卸载时清理 QuickSideNote 开机启动项，保留用户便签和 API Key。
- 构建脚本校验版本与外部工具退出码，生成安装器、便携 ZIP 和 SHA-256。
- 优化副显示器定位和便签文件恢复。

## v1.5.0 - 2026-07-08

- 合并 Z.ai / GLM 5.2 辅助优化后的 Quick Side Note UI。
- 主便签窗口改为暖色纸张质感，并将多页便签页签调整为顶部横排。
- 底部操作栏新增保存、清空、隐藏和设置入口。
- 设置窗口继续使用左侧导航，并为输入页保留滚动内容区，改善小窗口下文字显示不全的问题。
- 保留隐藏消息窗口托盘实现，右键托盘菜单不会回退到曾导致卡顿退出的旧路径。
- 版本号、安装器脚本、随包 README、介绍 HTML 和兼容性文档统一到 v1.5.0。
