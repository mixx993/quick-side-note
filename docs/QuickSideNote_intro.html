Quick Side Note 侧键便签

这是一款为阅读英文资料设计的轻量桌面工具：按下鼠标侧键即可框选屏幕文字， 程序自动识别、翻译，并把结果写入当前便签页和生词本。

当前版本 v1.4.3 面向日常使用的 Windows 安装版

本版重点修复托盘右键菜单崩溃问题，并保留侧键框选、翻译、便签和首次配置流程。

托盘稳定性 托盘事件由独立隐藏消息窗口接收，不再改写 Tk 主窗口过程。
右键菜单 右键打开显示/隐藏、设置、退出菜单；双击托盘图标快速显示或隐藏。
数据不迁移 升级安装只覆盖程序文件，便签和 API 配置仍保存在当前 Windows 用户目录。

围绕阅读和摘录做减法

软件把截图、识别、翻译、记录和词汇沉淀串成一个动作，减少在多个窗口之间切换。

1

侧键触发框选

单击鼠标侧键 1 进入屏幕框选，适合网页、PDF、图片和桌面软件里的英文片段。

2

本地 OCR 识别

优先使用 Windows OCR 在本机识别文字，避免把截图先交给远端模型处理。

3

DeepSeek 快速翻译

识别出的文本直接发送到 DeepSeek 文本接口，翻译链路比整图大模型处理更短。

4

多页便签

不同主题可以放到不同便签页，每页对应一个 txt 文档，便于长期归档。

5

生词本沉淀

每次识别和翻译会记录到 vocabulary.jsonl，后续可以导入复习工具或数据表。

6

托盘和设置中心

通知区域图标提供常驻入口，设置界面统一管理 API Key、开机启动、侧键选择和浏览器键拦截。

一次侧键完成取词到入库

默认路径适合高频英文阅读，失败时仍保留兜底能力。

01 按下侧键 进入屏幕框选状态。
02 框选英文 拖拽选中要翻译的区域。
03 本地识别 Windows OCR 提取文本。
04 接口翻译 DeepSeek 返回中文释义。
05 写入便签 结果保存到 txt 和生词本。

快捷键和侧键

  • 鼠标侧键 1 单击进入屏幕文字框选，识别并翻译英文。
  • 鼠标侧键 1 双击显示或隐藏便签窗口。
  • 框选界面再次按 侧键 1取消当前框选。
  • 托盘图标双击显示或隐藏便签窗口。
  • 托盘图标右键打开显示/隐藏、设置和退出菜单。
  • 框选界面按 Esc取消当前框选。
  • Ctrl + K打开 API 设置。
  • Ctrl + ,打开完整设置界面。
  • Ctrl + S保存当前便签。
  • Ctrl + Q退出程序。
  • Alt + 鼠标左键拖动移动便签窗口。

数据保存在用户文档目录

应用没有复杂数据库，便签、词汇、状态和日志都落在当前用户的 Documents 目录下。

默认文件

%USERPROFILE%\Documents\QuickSideNote\note.txt第 1 页便签
%USERPROFILE%\Documents\QuickSideNote\note-2.txt第 2 页及后续便签
%USERPROFILE%\Documents\QuickSideNote\vocabulary.jsonl生词和翻译记录
%USERPROFILE%\Documents\QuickSideNote\state.json窗口和当前页状态
%USERPROFILE%\Documents\QuickSideNote\quick_note.log运行日志

作为应用程序交付

推荐使用安装器部署到新电脑。安装完成后第一次启动会引导配置 API，后续可以在设置界面调整开机启动和侧键行为。

1. 运行安装器

双击 QuickSideNote_Setup_v1.4.3.exe，按向导安装到当前 Windows 用户目录。

2. 首次配置

第一次运行如果没有检测到 DEEPSEEK_API_KEY，会弹出首次设置窗口引导填写。

3. 开始阅读

打开网页、PDF 或图片资料，按鼠标侧键框选英文；程序会自动识别、翻译并写入便签。
