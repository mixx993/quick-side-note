Quick Side Note
侧键便签

这是一款为阅读英文资料设计的轻量桌面工具：按下鼠标侧键即可框选屏幕文字， 程序在本地识别、翻译并记录结果，整个任务过程始终有状态反馈和取消入口。

查看操作方式 首次安装配置
过程始终可见 框选、识别、翻译和完成都会显示清晰状态。
失败可恢复 再次按侧键或 Esc 可取消；失败后可重新框选或返回便签。
即时或兼容 可保留双击切换，也可启用即时框选，消除单击等待。
便签窗口预览

暖色纸张窗口支持多页切换、自动保存、清空撤销和固定操作栏。

当前版本 v1.5.2 界面可用性与交互闭环更新

本版修复底部操作栏和设置页裁切，增加翻译任务状态、失败恢复、清空撤销和即时框选模式。

任务状态 任务 HUD 覆盖准备框选、识别、翻译、完成、取消和失败恢复。
误操作保护 清空当前页后可在 8 秒内撤销，不会立即覆盖原有内容。
设置更完整 API 可测试连接，所有设置页可滚动，并会适应当前显示器工作区。

围绕阅读和摘录做减法

软件把截图、识别、翻译、记录和词汇沉淀串成一个动作，减少在多个窗口之间切换。

1

侧键快速框选

单击侧键即可框选网页、PDF、图片和桌面软件中的英文片段。

2

本地 OCR 优先

优先使用 Windows OCR 在本机识别文字，截图兜底默认关闭。

3

状态化翻译

识别和翻译阶段都有独立状态面板，等待过程不再像程序无响应。

4

多页便签

不同主题放在不同 txt 页面；页签提供更多菜单和键盘重命名入口。

5

清空可撤销

误清空当前页后可在 8 秒内恢复，保留记录时的连续感。

6

托盘和设置中心

管理 API 测试、开机启动、侧键模式和浏览器快捷键拦截。

一次侧键，状态一路可见

v1.5.2 把框选后的等待变成可理解、可取消、可恢复的连续流程。

01 按下侧键 兼容模式等待双击；即时模式立即框选。
02 准备框选 任务面板提示当前操作，再次按侧键或 Esc 可取消。
03 框选英文 拖拽选中要翻译的区域。
04 本地识别 Windows OCR 优先提取文本。
05 文本翻译 DeepSeek 返回简明中文释义。
06 写入便签 结果写入当前 txt 页面和生词本，失败时可重新框选。

不让等待、失败和误触打断阅读

工具应该把不确定的后台过程变成用户能看懂、能控制的界面状态。

任务状态 HUD

主窗口隐藏后，准备框选、正在识别和正在翻译仍然会在屏幕上显示，不需要猜测程序是否卡住。

取消与失败恢复

侧键和 Esc 都能取消当前任务。网络、API 或识别失败后，可以重新框选，也可以直接返回便签。

保存不冒险

输入自动保存；清空当前页后保留撤销窗口。便签和状态继续使用原子写入与备份恢复路径。

常用操作

按键设计围绕少打断阅读节奏：侧键负责取词，快捷键负责管理窗口和配置。

快捷键和侧键

  • 鼠标侧键 1 单击进入屏幕文字框选，识别并翻译英文。
  • 鼠标侧键 1 双击显示或隐藏便签窗口。
  • 即时框选模式侧键单击立即开始框选；窗口切换改用托盘或 Ctrl+Alt+N。
  • 任务中再次按 侧键 1 或 Esc取消框选、识别或翻译任务。
  • 托盘图标双击显示或隐藏便签窗口。
  • 托盘图标右键打开显示/隐藏、设置和退出菜单。
  • Ctrl + K打开 API 设置。
  • Ctrl + ,打开完整设置界面。
  • Ctrl + S保存当前便签。
  • Ctrl + L清空当前页；8 秒内可以撤销。
  • Ctrl + Alt + N显示或隐藏便签窗口。
  • Ctrl + Q退出程序。
  • Alt + 鼠标左键拖动移动便签窗口。

数据保存在用户文档目录

应用没有复杂数据库，便签、词汇、状态和日志都落在当前用户的 Documents 目录下。

默认文件

%USERPROFILE%\Documents\QuickSideNote\note.txt第 1 页便签
%USERPROFILE%\Documents\QuickSideNote\note-2.txt第 2 页及后续便签
%USERPROFILE%\Documents\QuickSideNote\vocabulary.jsonl生词和翻译记录
%USERPROFILE%\Documents\QuickSideNote\state.json窗口和当前页状态
%USERPROFILE%\Documents\QuickSideNote\quick_note.log运行日志

作为应用程序交付

推荐使用安装器部署到新电脑。安装完成后第一次启动会引导配置 API，设置页可测试连接，并管理开机启动和侧键响应模式。

1. 运行安装器

双击 QuickSideNote_Setup_v1.5.2.exe，按向导安装到当前 Windows 用户目录。

2. 首次配置

第一次运行如果没有检测到 DEEPSEEK_API_KEY，会弹出首次设置窗口；保存前可测试 API 连接。

3. 开始阅读

打开网页、PDF 或图片资料，按鼠标侧键框选英文；任务状态会持续显示，结果自动写入便签。
